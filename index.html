How to use webscope_to_openseadragon.js

  1. Navigate to a webpage with an Aperio WebViewer whole slide image (.svs file)
  2. You will probably see a Flash Player error unless you've enabled Flash for the site previously
  3. Open the Developer Tools in your browser (F12)
  4. In the console window, paste the following code: 

            var s = document.createElement('script');s.src="https://pearcetm.github.io/webscope-to-openseadragon/webscope_to_openseadragon.js";document.head.append(s);
